# Programming Fundamentals

This repository contains all lab exercises for the **Programming Fundamentals** course.  
Labs are written in **C++** and cover basic programming conceptS.


---

## Lab Descriptions

### **Lab 1 — Introduction & Basic Output**
**Objective:**  
- Get familiar with Dev-C++ IDE.  
- Understand basic structure of a C++ program.  
- Perform simple input/output using `cout`.  

**Tasks:**  
1. **Simple Welcome Message:** Display a welcome message.  
2. **Printing a Pattern:** Print a simple text pattern using multiple lines.  
3. **Formatting with Tabs and Newlines:** Use `\t` and `\n` to format text.  
4. **Message Using Various Escape Sequences:** Use `\n`, `\t`, `\"`, and `\\` in a formatted message.  

**Learning Outcomes:**  
- Successfully installed and configured a C++ environment.  
- Learned basic text output and formatting.  

---

### **Lab 2 — Using Constants**
**Objective:**  
- Learn how to declare and use constant variables in C++.  

**Key Concepts:**  
- Constant variables (`const`)  
- Basic arithmetic operations  
- Input/output using `cin` and `cout`  
- Syntax correctness and readability  

---

### **Lab 3 — Comments & Data Types**
**Objective:**  
- Build upon Lab 2 with comments and fundamental data types.  

**Key Concepts:**  
- Single-line (`//`) and multi-line (`/* */`) comments  
- Data types: `int`, `float`, `char`, `double`, `bool`  
- Program structure and readability  

---

### **Lab 4 — Ternary Operator Tasks**
**Objective:**  
- Practice using the ternary operator for decision making.  

**Tasks:**  
1. **Even or Odd:** Determine if a number is even or odd.  
2. **Age Group Classification:** Classify a person’s age: Child (<13), Teenager (13–19), Adult (20+).  
3. **Banking App:** Determine account type and special offer eligibility based on balance and loyalty.

**Learning Outcomes:**  
- Understand and apply the ternary operator in real scenarios.  
- Improve logical thinking and C++ coding skills.  

---

## **Technologies Used**
