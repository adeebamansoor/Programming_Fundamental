### **Task 3: Input Validation Using While Loop**
This program repeatedly asks the user to enter a **positive number**.  
If the user enters **zero or a negative number**, an error message is displayed and input is requested again.

The program terminates only when a valid positive number is entered.

#### Concepts Used:
- While loop
- Input validation
- Conditional statements

#### Concepts Used:

### **Task 5: Menu-Driven Banking System**
This is a **menu-driven program** for a simple banking system.  
The system starts with a **default balance of $1000** and continues to run until the user exits.

#### Menu Options:
1. Deposit Money  
2. Withdraw Money  
3. Check Balance  
4. Exit  

#### Features:
- Prevents withdrawal if balance is insufficient
- Displays error messages for invalid choices
- Uses while loop to keep showing the menu

---

### **Task 6: Password Authentication System**
This program prompts the user to enter a **password**.

#### Concepts Used:
- While loop
- String comparison
- Authentication logic

### **Task 7: Restaurant Ordering System**

#### Concepts Used:
- Nested menus
- While loop

- Real-world problem solving
